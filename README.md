# D3-challenge - Data Journalism and D3

## Homework Submission:

The structure of this repository is as followed: 

1- Main (published) file: _index.html_ 

2- Under folder "assets":

  - Sub folder css: contains styling files, _d3Style.css_ and customised _style.css_
  - Sub folder data: contains input data file _data.csv_
  - Sub folder js: contains javascript _app.js_ (including bonus section)
    
**Note:** 
- The circles will be changing the filling/ background color to be more distinct when mouseover and mouseoff.
- Some observations for data analytics are also performed in the html body text.
- VSCode/ python live server is used to run the data visualisation.
- URL of github page: https://gigi8android.github.io/D3-challenge/
